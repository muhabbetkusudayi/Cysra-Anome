# Cysra Anome 7.0

Cysra Anome 7.0 brings a cleaner interface and smoother browsing experience. The UI has been refined, rendering issues have been fixed, and new personalization utilities were added.

## What's New

- UI fixes and visual stability improvements  
- History system added  
- Favorite pages feature added  
- Rendering engine issues resolved  

## Improvements

- Fixed layout glitches across pages  
- Optimized component rendering  
- Faster navigation between panels  
- More consistent styling behavior  

## Usage

Download the latest release and replace your existing application files with the new version.

If you are upgrading from an older version, make sure to back up your data before installing.

## Notes

- History and Favorite data are stored locally unless cloud sync is enabled in future updates.  
- Some extensions or custom scripts may affect rendering behavior.
